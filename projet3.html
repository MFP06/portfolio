Configuration d’un site web sur un sujet personnel

Projet orienté web

Dans le cadre de notre formation, nous avons travaillé en binôme pour créer un site web statique en HTML et CSS. Ce projet avait pour but de nous initier aux bases du développement web tout en abordant un sujet qui nous passionne : le sport. Nous avons conçu une interface simple et claire, mettant en avant les bienfaits de l’activité physique, ainsi que différentes disciplines sportives. Ce travail nous a permis de mieux comprendre la structure d’un site, la mise en page, et l’importance de l’ergonomie dans la présentation de contenus.

Ce travail nous a permis de comprendre les bases de la programmation web.

Ici vous pouvez trouver un lien menant vers le github ou vous trouverez le code que j'ai créer.

Compétences mises en œuvre

  • Developpement : HTTP, CSS
  • Collaboration
